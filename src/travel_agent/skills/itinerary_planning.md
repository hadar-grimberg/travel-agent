# Itinerary Planning Skill

Build a day-by-day plan that feels personal, realistic, and enjoyable — not a packed checklist.

## Structure per day
- **Theme** — one-line focus (e.g., "Old Town & riverfront")
- **Morning / Afternoon / Evening** — 1–2 items each; allow breathing room
- **Meals** — specific restaurant or area + backup option
- **Driving notes** — when day trips or cross-town moves apply

## Pacing by travel style
| Style   | Active hours/day | Notes                          |
|---------|------------------|--------------------------------|
| relaxed | 4–6              | Long lunches, flexible blocks  |
| balanced| 6–8              | One anchor + one flexible slot |
| packed  | 8–10             | Early starts; pre-book key slots |

## Length-based guidance
- **1–2 days:** hit signature experiences only; minimize transit.
- **3–4 days:** blend icons + one day trip if budget allows.
- **5–7 days:** include 1–2 slower days and a day trip.
- **8+ days:** weekly rhythm — intense / moderate / recovery pattern.

## Preferences
- Honor dietary restrictions in every meal suggestion.
- Weight interests explicitly (food-heavy vs nature-heavy itineraries differ).
- Stay within daily budget derived from total budget ÷ trip days.
